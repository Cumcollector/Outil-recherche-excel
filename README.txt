Mode d'emploi :

1 - Cliquer sur le bouton "Statistiques matériel" pour ouvrir l'outil de recherche

2 - Sélectionner le fichier contenant l'étude

3 - Sélectionner le fichier contenant l'arborescence

4 - Choisir la pièce

5 - Cliquer sur rechercher

6 - cliquer sur fermer pour pouvoir effectuer des opérations sur les graphiques (copier, couper ...) ou lancer un nouvelle recherche


PS : les fichiers d'étude et d'arborescence doivent être de la même forme que les fichiers exemples présent dans le dossier (même emplacement de colonne pour les différentes catégories).
Le taux de panne est une moyenne depuis 2019 du taux annuel de panne de la pièce (nombre réparation / ( nombre de ram total * nombre de piece par ram))

ATTENTION : ne pas supprimer le bouton et les cases contenant les villes (sinon, fermer le fichier sans enregistrer)
